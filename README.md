##Instalation
###Docker
```
git clone https://github.com/BlackthornYugen/websocks.git
cd websocks
sh ./build.sh
docker run -d -p 8000:8000 dev_one
```
###NPM
```
git clone https://github.com/BlackthornYugen/websocks.git
cd websocks
npm install
npm start
```
